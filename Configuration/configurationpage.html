Dynamic Playlists

Define unlimited smart playlists. Each playlist is rebuilt automatically by the Collection Manager scheduled task based on the filters you configure here.

Binge Playlists

When checked, the Collection Manager automatically creates binge-watch playlists for movie franchises (e.g. "The Hobbit Franchise") in your library. Uncheck to opt out — any existing "* Franchise" playlists you have will be removed on the next task run. The server administrator must also have this feature enabled.
When checked, the Collection Manager automatically creates binge-watch playlists for TV universes and spin-offs (e.g. "Game of Thrones Universe"). Uncheck to opt out — any existing "* Universe" playlists will be removed on the next task run. The server administrator must also have this feature enabled.

Smart Playlists

When checked, the playlist is rebuilt by the scheduled task. When unchecked, any existing playlist with this name will be removed on the next run (the configuration is kept so you can re-enable it later).
