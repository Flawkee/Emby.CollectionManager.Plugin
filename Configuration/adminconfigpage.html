Collection Manager

Saving this page triggers an immediate run of the Collection Manager scheduled task. The task processes phases in this order: Dynamic User Playlists → Movie Franchises → TV Universes → Streaming Collections.

Disabling a feature here turns it OFF for every user and removes any matching playlists they already have. When a feature is enabled here, each user can still opt out individually from their personal Dynamic Playlists page (top-right user menu → Dynamic Playlists).

Dynamic Playlists

When enabled, each user can define their own playlists from the user menu → Dynamic Playlists page, with filters for genre, studio, year, parental rating, tags, play state, favorites and series status. Playlists are rebuilt at every task run and after each user save. When disabled, the scheduled task removes every user's existing dynamic playlists. Binge playlists (Movie Franchises / TV Universes) are NOT affected — they have their own toggles below.

Movie Franchises

When enabled, the task creates a private "<Franchise> Franchise" playlist for each user (e.g. "The Hobbit Franchise", "Pirates of the Caribbean Franchise"). Each user can opt out from their Dynamic Playlists page. When disabled, every "* Franchise" playlist is removed for every user on the next task run.
When provided, franchises are detected via The Movie Database collection data — the most accurate grouping (e.g. Lord of the Rings, Avatar, Mission: Impossible). Without a key, movies are grouped by shared name prefix (e.g. "Avatar" + "Avatar: The Way of Water" → "Avatar Franchise"). Get a free v3 key at themoviedb.org → Settings → API → Request an API Key → Developer.

TV Universes

When enabled, the task creates a private "<Universe> Universe" playlist for each user grouping TV shows with franchise relationships from TVMaze (e.g. "Game of Thrones Universe", "Breaking Bad Universe"). Only Franchise / Prequel / Sequel / Spin-off / Companion Series relationships are included — After Shows and talk shows are excluded. Each user can opt out from their Dynamic Playlists page. When disabled, every "* Universe" playlist is removed for every user on the next task run.
Sets a custom poster on the Playlists library entry so it stands out in Home. Disable to manage that image manually.

Streaming Collections

When enabled, the task creates a public collection per streaming service detected in your library's studio metadata (Netflix, Amazon Prime Video, Disney+, Hulu, HBO Max, Apple TV+, Paramount+, MGM+, YouTube TV, Sling TV, Discovery+, ESPN+). Items are grouped by the streaming service name on their studio field. Collections are shared across users (not per-user). When disabled, every collection whose name matches one of the managed streaming services is removed on the next task run.
Sets a custom poster on the Collections library entry. Disable to manage it manually.

Scheduled Collections

Builds public collections only when their schedule is active, then removes them when inactive if RemoveWhenInactive is true. Use this for ACDB-style seasonal collections such as Halloween in October or Action Night every Friday.

Diagnostics

Writes verbose step-by-step logs for every operation. Useful when a playlist or collection isn't being created the way you expect. Disable once the issue is resolved to keep logs clean.
